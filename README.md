# 🍡 Mochi Space - Your Cozy Self-Care Companion

A beautiful, interactive landing page for Mochi Space, an adorable self-care app designed to help you track moods, practice mindfulness, and build healthy habits in the coziest way possible.

## ✨ Features

### Landing Page Sections

1. **Hero Section**
   - Eye-catching headline with gradient text
   - Cute Mochi character animation
   - Interactive phone mockup showcasing app interface
   - Floating elements (hearts, stars, clouds) with smooth animations
   - Call-to-action buttons

2. **Features Section**
   - 6 main features with interactive cards:
     - 😊 Mood Tracking
     - 🌬️ Breathing Exercises
     - 📔 Gratitude Journal
     - 🌙 Sleep Sounds
     - 💝 Daily Affirmations
     - ✅ Habit Tracker
   - Feature cards with hover effects and visual previews

3. **How It Works**
   - Simple 3-step process explanation
   - Easy-to-follow journey description

4. **App Preview**
   - Interactive phone mockups showing different app screens
   - Beautiful screen previews with animations

5. **Testimonials**
   - User reviews and testimonials
   - 3D rotation effect on hover
   - Authentic user feedback

6. **Download CTA**
   - Eye-catching gradient background
   - Animated Mochi character
   - App store download buttons (Apple App Store & Google Play)

7. **Footer**
   - Company information and social links
   - Quick navigation links
   - Support and legal links

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server setup required - works with static HTML/CSS/JS

### Installation

1. Clone the repository:
```bash
git clone https://github.com/gracianoya26-design/mochispace-cutie.git
cd mochispace-cutie
```

2. Open the landing page:
```bash
# Simply open index.html in your browser
open index.html

# Or use a local server (optional)
python -m http.server 8000
# Then visit http://localhost:8000
```

## 🎨 Customization

### Change Colors
Edit the CSS variables in `css/tooplate-mochi-space.css`:

```css
:root {
  --primary-color: #ff6b9d;      /* Your primary color */
  --secondary-color: #c44569;    /* Your secondary color */
  --accent-color: #ffa502;       /* Your accent color */
  /* ... more variables ... */
}
```

### Add Content
- Edit `index.html` to add more features or testimonials
- Update text, emojis, and descriptions as needed
- Add new sections by duplicating existing patterns

### Modify Animations
- Adjust timing in `css/tooplate-mochi-space.css` keyframes
- Change animation delays and durations
- Modify transitions for different effects

## 📞 JavaScript Functionality

The `js/tooplate-mochi-script.js` file includes:

1. **Mobile Menu**: Toggle-able mobile navigation
2. **Phone Clock**: Real-time clock display with 12/24 hour toggle
3. **Dynamic Greetings**: Time-based greeting messages
4. **Scroll Animations**: Reveal elements as they come into view
5. **Hover Effects**: Interactive card and testimonial animations
6. **Smooth Scrolling**: Anchor link navigation
7. **Parallax Effect**: Floating shapes move with scroll
