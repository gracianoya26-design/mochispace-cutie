🌤️ Weather Dashboard - Setup Guide

Follow these simple steps to get your weather dashboard up and running!

📋 Quick Setup (3 Minutes)

1Get Your Free API Key

OpenWeatherMap provides free weather data. Let's get your API key:

  1. Visit https://openweathermap.org/api
  2. Click "Sign Up"
  3. Fill in the registration form:
    • Email
    • Password
    • Confirm Password
  4. Verify your email (check inbox)
  5. Login to your account
  6. Go to "API Keys" section (usually in the menu)
  7. Copy the Default API Key

2Set API Key in Browser

Now let's add your API key to the dashboard:

  1. Open the Weather Dashboard
  2. Press F12 or Right-click → Inspect
  3. Click on the "Console" tab
  4. Paste this command (replace YOUR_API_KEY with your actual key):
    localStorage.setItem('weatherAPIKey', 'YOUR_API_KEY_HERE'); location.reload();
  5. Press Enter
  6. Page will reload automatically

Example:

localStorage.setItem('weatherAPIKey', 'sk-a1b2c3d4e5f6g7h8'); location.reload();

3Test It Out!

Your dashboard is now ready:

  1. Type a city name (e.g., "London", "Tokyo", "New York")
  2. Press Enter or click Search
  3. Weather data should load! 🎉

📱 Features to Try

  • 🔍 Search: Type any city name to get weather
  • 📍 My Location: Click the location button to use your GPS
  • ❤️ Favorites: Click heart to save favorite cities
  • 📊 Forecast: View 5-day forecast and hourly updates
  • 💨 Air Quality: Check air quality index and pollutants

⚙️ Advanced Setup (Optional)

If you want to set the API key permanently:

  1. Edit js/weather-api.js
  2. Find this line:
    const apiKey = localStorage.getItem('weatherAPIKey') || 'YOUR_OPENWEATHERMAP_API_KEY';
  3. Replace with your key:
    const apiKey = localStorage.getItem('weatherAPIKey') || 'sk-your-actual-key';

❓ Troubleshooting

Q: "API Key Invalid" error?

  • Check that you copied the key correctly (no extra spaces)
  • Make sure you verified your email on OpenWeatherMap
  • Try generating a new API key
  • Wait 5 minutes after account creation before first use

Q: Location button doesn't work?

  • Check if location permission is enabled in browser settings
  • Allow location access when browser asks
  • Try a different browser

Q: "City not found" error?

  • Try different spelling
  • Use full city names (e.g., "San Francisco" instead of "SF")
  • Include country code (e.g., "Paris, FR")
